shrkbot is a modular Discord bot. Enable just the plugins your server needs — nothing more, nothing less. Configure everything in one dashboard.

Add shrkbot to your server View on GitHub

Available plugins

Roles

Let members self-assign roles from a tidy menu you post in a channel.

Welcomes

Greet new members with a custom message, and say goodbye when they leave.

Logging

Record moderation events to a private staff channel — role changes, joins, leaves.

Reminders (/remind) are always available — no setup needed.
